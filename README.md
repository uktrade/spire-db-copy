# spire-db-copy

**Dummy App for connecting to SPIRE DB - the Department for International Trade (DIT)**

---

## Useage

First SSH into the app

cf ssh spire-db-copy


### Connect via Oracle Client:
```
~/oracle/instantclient_21_5/sqlplus READUSER@spire
```

### Connect via Python:
```
./test_connection.py
```
